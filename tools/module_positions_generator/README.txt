This program can generate a file containing the global coordinates of the tracker's modules. This file is needed for the global to local convertion in the AMSimulation tool (modules_position.txt)).
The input is a Root file containing a StripCoords TTree. This file can be generated using the method described in part 3.1.5 of http://sviret.web.cern.ch/sviret/Welcome.php?n=CMS.HLLHCTuto620 .
